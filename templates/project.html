Machine Learning using Python

This project is all about creating a Model that detect the types of Iris plant. This Model is created in Jupyter Notebook using Scikit-learn module which comes with built-in Iris dataset. Other than that Joblib module is also imported in this project to persist a scikit-learn model for future use. Nearest neighbors classifier: KNeighborsClassifier is also imported to create the Model.

Password Checking App using Python

This app will check for number of times a password has been hacked by receiving a password as an input. The program has been created by importing few modules such as requests and hashlib. The output produced will be in a text file due to security reasons. Multiple passwords can also be inputed at a time by using sys.argv[1:] in the program the output of which will be displayed in Command Prompt.

Web Scraping using Python

This app will scrape the data from an existing website to make decisions for our day-to-day life. The program is created using requests module to get the url of the website. Other than that beautifulSoup4 is installed and imported to create an object that can be used to manipulate html data of the website. pprint module is also imported. The output of the program is displayed in Command Prompt.
